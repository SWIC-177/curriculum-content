## Tailwind CSS

I barely remember how to write 'vanilla CSS ğŸ’„.' With all of the braces and semicolons and switching around between HTML and CSS files...nope, couldn't be me! Enter Tailwind.

Tailwind doesn't replace knowledge of CSS ğŸ’„. You still need to know it and how it works. But, it does save the tedium and time of writing it out from scratch.

It's a utility-first CSS framework. This means that it's designed to be used directly in your HTML. You don't have to switch back and forth between HTML and CSS files. You can just write your styles right in your HTML.
